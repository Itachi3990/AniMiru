## Prerequisites

Before running this app, ensure you have completed the [React Native environment setup](https://reactnative.dev/docs/set-up-your-environment).

## Installation

Install the dependencies:

```sh
npm install
```

### iOS Setup (macOS only)

Install Ruby dependencies:
```sh
bundle install
```

Install CocoaPods dependencies:
```sh
cd ios && bundle exec pod install && cd ..
```

## Running the App

### Start Metro

Start the Metro bundler:

```sh
npm start
```

### Run on Android

In a new terminal:

```sh
npm run android
```

### Run on iOS

In a new terminal:

```sh
npm run ios
```
